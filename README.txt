== Description ==

Wolfeo is an online software which helps you get more trafic, leads and sales with marketing automation tools.

Pushy is the Push Notification tool that helps entrepreneurs, bloggers and agencies collect leads and generate more sales online.

Learn more at https://wolfeo.net/.


Setup :

1. Install the Pushy by Wolfeo - Wordpress plugin on your wordpress website.
2. Get your API KEY from the API area of your Wolfeo dashboard: https://www.wolfeo.me/api
3. Add your API KEY in the setup page of the plugin, and save the selection of your default campaign.
4. Select the pages where you want to show your custom Invitation to Push Notifications by Pushy.
5. Enjoy the marketing automation :-)


Support :

You can contact us here at support@wolfeo.net.

== Installation ==

1. Download the wolfeo-pushy plugin from the Wordpress repository.
2. Unzip the downloaded zip file, and upload the folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Edit the settings for the plugin in the 'Settings' > 'Wolfeo Pushy' menu, and follow the setup steps in the description above.

== Frequently Asked Questions ==

Where do I get my API key from?
You can retrieve your unique key from your Wolfeo admin panel, under the API panel in the sidebar.
